# Интерактивная карта Антарктики
Образовательный проект призванный помочь школьникам и студентам в познавании южного материка.

## Использование
1. Установите python версии 3.13
2. Скачайте проект
3. Откройте проект и установите все зависимости
`pip install -r requirements.txt`
4. Запустите файл PY.py
